# So_Long

## Description

A 2D tile-based game in C using MinilibX. Collect all items and reach the exit while avoiding walls. Move counter tracks progress.

## Instructions

**Compilation:**
```bash
make
```

**Execution:**
```bash
./so_long maps/valid1.ber
```

**Controls:** W/A/S/D or Arrow Keys to move, ESC to exit.
